Universidade Livre

Resumo:
Computadores estão presentes em todos os lugares: em casa, nas escolas, nas universidades, nas empresas e até mesmo incorporados a dispositivos portáteis. Considerando que nem sempre é possível ou acessível adquirir software para serem utilizados nesses computadores por diversos motivos, entre eles pode-se citar: o preço das licenças de utilização que muitas vezes é alto, as limitações de utilização e redistribuição do software adquirido, considerando que cada licença é exclusiva para um único computador.
Considerando ainda a grande expansão das comunidades especializadas que desenvolvem, debatem e divulgam as propostas e a ideologia do software livre, neste projeto é proposto a disseminação de software livre utilizando como ponto de partida a Universidade Estadual de Alagoas – UNEAL Campus III, localizada no município de Palmeira dos Índios/AL. Durante a realização deste projeto serão avaliados os equipamentos disponíveis e o software utilizado em tais máquinas. Com base nesse levantamento, pretende-se realizar a substituição do software proprietário para um software livre equivalente. Buscando minimizar as dificuldades relacionadas às mudanças no ambiente de software, serão oferecidos cursos de capacitação para os usuários da UNEAL.
