O xadrez como ferramenta de recreação e desenvolvimento intelectual do cidadão

Resumo: Este trabalho teve como objetivo auxiliar o aprendizado e aprimoramento do xadrez, que é esporte, arte e ciência, por meio de uma ferramenta de software. Ferramenta esta que proporciona ao interessado em xadrez maior autonomia no processo de aprendizado. Tal ferramenta foi implementada usando a linguagem de programação Python por se tratar de uma linguagem portátil. A ferramenta possui licença GPL (General Public License), sendo assim software livre.
