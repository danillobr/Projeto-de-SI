Programação Java utilizando a IDE Eclipse

Resumo: A tarefa de elaborar um software de computador é um tanto complexa e requer organização e raciocínio. A interface de desenvolvimento Eclipse facilita o trabalho do programador, atuando na organização e correção de código, compilação, organização de pacotes etc. Além disso, alguns programadores não conhecem ainda a linguagem Java ou mesmo o paradigma de orientação à objetos. Dessa forma, este mini-curso terá como objetivo ensinar conceitos básicos da linguagem de programação Java, utilizando a IDE Eclipse. As aulas serão teóricas e práticas, ambos os modos no laboratório de informática.
